CODES COULEURS DE SÉCURITÉ

CODE BLANC
Situation normale. Opérations de routine. Aucune menace détectée.
CODE VERT
Incident mineur sous contrôle. Pas de risque de propagation ou de brèche.
CODE BLEU
Urgence médicale ou technique. Intervention d'une équipe spécialisée requise.
CODE JAUNE
Menace potentielle détectée. Vigilance accrue. Préparation à un possible confinement.
CODE ORANGE
Brèche de confinement mineure ou intrusion suspectée. Déplacement restreint.
CODE ROUGE
BRÈCHE DE CONFINEMENT MAJEURE. Combat engagé. Évacuation immédiate des secteurs non-combattants.
CODE NOIR
DÉCONFINEMENT TOTAL / SCÉNARIO DE CLASSE K. Destruction du site imminente. Protocole "Last Stand".

CODES FONTAINE (SPÉCIFIQUES)

SIGNAL DELTA
Alerte radioactive. Utilisation de compteurs Geiger et blindage plombé.
SIGNAL ECHO
Phénomène mémétique ou cognitif détecté. Utilisation de filtres visuels et protections auditives.
SIGNAL X-RAY
Entité invisible ou déphasée détectée. Sortie des scanners thermiques et radars à ondes courtes.
