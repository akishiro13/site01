CONCORDANCE AVEC LES PROTOCOLES OFFICIELS DE LA FONDATION // RÉVISION 04-B

⚠ CLASSIFICATION DE SÉCURITÉ MÉDICALE

Ce document récapitule les classes officielles d'amnésiques synthétiques et non-anormaux. Toute administration hors protocole ou sur du personnel non-autorisé est passible de sanctions de Niveau 4.

Rétrograde Général CLASSE A
APPLICATION : Aérosol / Oral
CIBLE : 5-6 Heures
L'amnésique de Classe A est le standard de terrain. Il déconsolide les souvenirs par rupture de la fenêtre de reconsolidation. Très efficace pour les rencontres d'entités fortuites.
Standard O5-Council
Rétrograde Régressif CLASSE B
APPLICATION : Oral / Injectable
CIBLE : Linéaire
Efface les souvenirs en partant du plus récent vers le plus ancien. La profondeur de l'effacement est directement proportionnelle au dosage administré.
Rétrograde Cinglé CLASSE C
APPLICATION : Neuro-Stimu
CIBLE : Sélective
Utilisé avec l'imagerie neuronale pour cibler des engrammes spécifiques. Permet de supprimer une information précise (ex: un code secret) sans toucher au reste de la mémoire.
Rétrograde Progressif CLASSE D
APPLICATION : IV Dose-Haute
CIBLE : Passé Lointain
L'opposé de la Classe B. Cible les souvenirs les plus anciens en premier. Très risqué, peut causer des lésions cérébrales irréversibles. Rarement utilisé.
Ennui (Dissociation) CLASSE E
APPLICATION : Social
CIBLE : Émotionnelle
Affaiblit les liens émotionnels liés aux souvenirs de l'anormal. Le sujet finit par accepter l'anomalie comme "normale" ou sans importance jusqu'à son oubli naturel.
Fugue (Identité) CLASSE F
APPLICATION : Psychotrope
CIBLE : Identitaire
Provoque une amnésie dissociative totale. On l'utilise pour réinitialiser complètement l'identité d'un sujet afin de lui en donner une nouvelle (Classe D, ex-agents).
Gaslighting CLASSE G
APPLICATION : Inhalant
CIBLE : Réalité
Induit une déréalisation. Le sujet commence à douter de la véracité de ses souvenirs, les percevant comme des rêves ou des hallucinations fantaisistes.
Antérograde CLASSE H
APPLICATION : IV Continue
CIBLE : Futur
Empêche la formation de nouveaux souvenirs. Utilisé lors de transferts de détenus dangereux ou d'objets SCP info-sensoriels.

📂 Annexe : Les Mnésiques (Antidotes)

Utilisés principalement par le Département de l'Antimémétique pour contrer l'oubli forcé ou naturel.

Classe W : Amélioration générale de la mémoire. Capable de percer les voiles antimémétiques légers.
Classe Y : Rappel parfait. Le sujet se souvient de chaque détail avec une clarté photographique.
Classe Z : Incapacité biochimique d'oublier quoi que ce soit. Mortel à terme car le cerveau ne peut plus traiter l'afflux.

Protocoles de Terrain

  • ▪ Aérosol : Utilisation civile rapide. Dose standard 0.5mg/m3.
  • ▪ Voie Orale : Pour les interrogatoires prolongés. Action sous 15min.
  • ▪ Flash Visuel (Anormal) : Induit un court-circuit synaptique (non documenté ici).
